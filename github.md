## Описание
Современное десктопное приложение для учета личных финансов, разработанное на Python с использованием PyQt6 и PyQt-Fluent-Widgets. Приложение предоставляет интуитивно понятный интерфейс для отслеживания доходов и расходов, визуализации статистики и экспорта отчетов.

## Технологический стек
- **Язык**: Python 3.10+
- **GUI Фреймворк**: PyQt6
- **UI Библиотека**: PyQt-Fluent-Widgets (Windows 11 Fluent Design)
- **Графики**: PyQt6-Charts
- **База данных**: SQLite
- **Экспорт**: Pandas, OpenPyXL, ReportLab
- **Тестирование**: Pytest

## Функциональность
1. **Управление транзакциями**: Добавление, редактирование, удаление доходов и расходов.
2. **Хранение данных**: Локальная база данных SQLite.
3. **Просмотр**: Таблица транзакций с фильтрацией по дате и типу.
4. **Аналитика**:
    - Расчет текущего баланса.
    - Столбчатая диаграмма доходов/расходов по месяцам.
    - Круговая диаграмма по категориям.
5. **Экспорт**: Выгрузка данных в Excel и PDF.
6. **Интерфейс**: Современный дизайн (Fluent Design), светлая тема.
